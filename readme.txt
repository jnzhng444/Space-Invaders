Hola, estimado usuario/a:
Este es un juego creado con el fin de entretener.
Para poder jugar el juego se necesita descargar cualquier programa de editor de código, ya sea PyCharm, Visual Studio Code, entre otros.
Tiene que ser una versión de Python actualizada, preferiblemente la versión 3.11 que es la versión que se utilizó para la creación del juego.
Para jugar, verifica que dentro de la carpeta se encuentran las siguientes carpetas "BackgroundMenu", "Fondo" y "Sonido", sin esto no se podrá ejecutar correctamente el código.
En caso de no estar, porfavor contactar al desarrollador por medio de un correo electrónico.
Para correr el juego, nada más abre el .py llamado PROYECTO.py.
Una vez abierto, corres el código y a disfrutar.
No se necesita una computadora de alta gama para poder disfrutar de este juego.
